# Game Thẻ Bài Việt Nam — "Việt Sát"

## Cấu trúc dự án

```
game-vietnam-the-bai/
├── client/          # React + Vite (Giao diện người chơi)
├── server/          # Node.js (Server multiplayer - Phase 3)
├── shared/          # Logic dùng chung giữa client & server
└── docs/            # Game Design Document & tài liệu
```

## Chạy ở môi trường Development

```bash
# Client (giao diện game)
cd client
npm install
npm run dev
```

## Tech Stack
- **Client:** React + Vite + Vanilla CSS
- **Server:** Node.js + Colyseus (Phase 3)
- **Mobile:** Capacitor (Phase 5)
